### up_down_cast

업케스팅과 다운케스팅 예제

* 업 케스팅``(자식클래스 -> 부모클래스)`` : 형변환``(부모 클래스)``을 생략할 수 있다.

* 다운 케스팅``(부모클래스 -> 자식클래스)`` : 형변환``(자식 클래스)``를 무조건 명시해야 한다.
