# Class_relationship_1

* 상속에 대한 예문

	* 원은 좌표상에 존재한다.
	
	* 좌표는 x, y의 두 점이다.
	
	* x, y상에 원이 존재한다.
	
	* x, y : 점이다.
	
	* 원 : x, y, 반지름 을 가진다.
	
	* 사각형 : x, y 가로변, 세로변 을 가진다.
	
	* 공통부분 : x, y
	
	* 기능 : ``도형을 그리다``
	
	* 위의 내용을 상속, 관계로 표현을 하면
	
		* Class 좌표
		
		* Class 모양(추상적 의미)
		
		* Class 도형
		
			* 도형 is a 모양
			
			* 도형 has a 좌표
